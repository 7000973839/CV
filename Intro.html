What is HTML

  1. HTML stands for Hyper Text Markup Language
  2. Markup language foe creating Web pages
  3. Describes the structure of a Web page
  4. Consists of a series of elements.
  5. Elements tell the browser how to display the content.
  6. Elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.
